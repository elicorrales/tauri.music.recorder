If the GitHub repo is empty (or only contains the default README/license), from inside tauri.music.recorder:

cd tauri.music.recorder

git init

git add .
git commit -m "Initial commit"

git branch -M main

git remote add origin https://github.com/elicorrales/tauri.music.recorder.git

git push -u origin main

If git remote add origin says origin already exists, use:

git remote set-url origin https://github.com/elicorrales/tauri.music.recorder.git

If the GitHub repo already has commits (such as a README created on GitHub), pull first:

git pull origin main --allow-unrelated-histories

Resolve any conflicts if prompted, then:

git push -u origin main
